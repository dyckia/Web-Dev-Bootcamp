### 1. Separate css file (recommended)

`<link>` tag should be put in the `<head>`

```html
<head>
  <!-- should be in the head -->
  <link rel="stylesheet" href="app.css">
</head>
```



###  2. Inline css

```html
<h1 style="color: blue;">Blue H1</h1>
```



### 3. Style tag

```html
<head>
  <style type="text/css">
    h1 {
      color: blue;
    }
  </style>
</head>
```
